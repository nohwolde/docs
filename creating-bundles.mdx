## Configuration Guide

<AccordionGroup>
  <Accordion title="1. Bundle Details">
    **Name your bundle** (for your reference only)
    - Use descriptive names: "Summer BOGO Socks"
    - Include product and type
    - Add dates for seasonal offers
    
    **Choose bundle type**
    - Volume or BOGO
    - Cannot change type after creation without losing settings
  </Accordion>
  
  <Accordion title="2. Product Visibility">
    **Select products** that will show this bundle
    - Add one or more products
    - Widget appears on each product's page
    - Volume/BOGO require specific products (not "all products")
  </Accordion>
  
  <Accordion title="3. Configure Tiers">
    **Volume:** Set quantity and discount per tier
    - Start with 2-3 tiers
    - Tier 1: Usually no discount (baseline)
    - Tier 2: 10-15% off (make it "Most Popular")
    - Tier 3+: 15-20% off (make it "Best Value")
    
    **BOGO:** Set buy quantity and free quantity
    - Buy 1 Get 1 Free = 50% savings
    - Buy 2 Get 1 Free = 33% savings
    - Buy 3 Get 2 Free = 40% savings
    
    **Add badges** like "Most Popular" or "Best Value" to guide customers
    
    **Include free gifts** (optional) like samples or accessories
  </Accordion>
  
  <Accordion title="4. Discount Names">
    **What customers see at checkout**
    
    Templates available:
    - Volume: "10% off 2 items", "Buy 2, Save 10%"
    - BOGO: "Buy 2 Get 1 Free", "33% off when you buy 3"
    
    Use placeholders: `{qty}`, `{discount}`, `{buy}`, `{getFree}`, `{save}`
    
    Keep under 50 characters
  </Accordion>
  
  <Accordion title="5. Widget Styling">
    **Theme Presets:** 8 pre-designed color schemes
    - Indigo, Emerald, Rose, Amber, Purple, Slate, Crimson, Teal
    
    **Customize:**
    - Background and border colors
    - Text colors (primary, price, discount)
    - Badge colors and styles
    - CTA button text and colors
    - Border radius (0-24px)
    
    **Header toggle:** Show/hide "Buy more save more" text
    
    **BOGO variant mode:**
    - Per-item: Different size/color for each item
    - Single: Same variant for all items
    - Hidden: No variant selection (fastest)
  </Accordion>
  
  <Accordion title="6. Preview">
    **Live preview panel** updates as you configure
    - Switch between products
    - See exact customer view
    - Test before activating
  </Accordion>
</AccordionGroup>

## Save Options

<Tabs>
  <Tab title="Save as Draft">
    **Configuration saved but not live**
    - No discount rules created
    - Widget doesn't appear on site
    - Perfect for testing
  </Tab>
  <Tab title="Save & Activate">
    **Bundle goes live immediately**
    - Creates Wix discount rules
    - Widget appears after publishing site
    - Discounts apply at checkout
  </Tab>
</Tabs>
